locar-aframe examples

  1. Basic A-Frame: This is a simple example which sets a fake GPS location and four boxes close to that location, all in the page's HTML.
  2. A-Frame with JavaScript: shows how you can dynamically create content using the A-Frame API. When a GPS position is received, four boxes are created to the north, south, east and west of the position. Also allows the user to set a fake GPS location.
  3. A-Frame with API communication: shows how you can dynamically create content from a web API using the A-Frame API. Downloads nearby points of interest and shows them as markers (marker taken from main AR.js repository).
